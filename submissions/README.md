# Submissions inbox (approval gate)

Approved call-for-papers files go here. **Committing a CFP file into this
directory is the approval step** — the `update-submissions` workflow then
extracts it, publishes the file under `docs/cfps/`, adds/updates the
conference row, and removes the file from here.

Workflow: a CFP arrives via the site's "Submit your CFP" button (Tally form) →
you review it → drop the PDF/Word file here and commit → it goes live.

Supported: `.pdf`, `.docx`, `.txt`, `.md`.
